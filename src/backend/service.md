## 1.3 프로젝트 범위

본 프로젝트는 아래와 같은 기능을 포함한다.

- 필수 기능
    - webOS App 실행 및 App List 관리
    - 미디어 재생 / 볼륨 조절
    - 사용자 계정 로그인
    - 시계 / 날씨 표시
- 선택 / 심화 기능
    - 얼굴 인식 로그인
    - 캐릭터 ‘빈버드’ 인터렉션
    - 메모장 기능
    - 앱 즐겨찾기 기능
- 비기능 요구사항
    - CPU 사용률 30% 이하
    - Memory Leak 없음

## 1.4 기술 스택

- **Frontend**: Flutter
- **Backend**: Express
- **Database**: PostgreSQL
- **Infra**: AWS EC2
- **CICD**: Jenkins, Docker
- **Test**: Jest, flutter test
- **협업도구**: Github, Discord, Figma
